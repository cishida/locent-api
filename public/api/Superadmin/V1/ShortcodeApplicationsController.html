ShortcodeApplications [1.0]

Superadmin::ShortcodeApplications

index

Gets paginated list of all shortcode application

URL

GET /superadmin/shortcode_applications

Response Fields

  • applications Array<ShortcodeApplications>

    All shortcode applications

update_status

Updates status of shortcode application

URL

PUT /superadmin/shortcode_applications/:id/update_status

Required Parameters

  • id Integer

    The application id.
  • status String

    The application status. It could be “pending”, “approved” or “rejected”.

Response Fields

  • application ShortcodeApplication

    The updated application

assign_shortcode_to_organization

Assign a shortcode to an organization

URL

POST /superadmin/shortcode_applications/:id/assign_shortcode_to_organization

Required Parameters

  • id Integer

    The organization id.
  • shortcode String

    The shortcode.

Response Fields

  • organization Organization

    The updated organization
