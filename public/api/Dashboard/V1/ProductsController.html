Products [1.0]

Products

create

Creates new product

URL

POST /dashboard/products

Required Parameters

  • name String

    The product name
  • uid String

    The product's unique identifier
  • feature String

    The feature the product is being added to
  • keyword String

    The product's keyword
  • price BigDecimal

    The product's price

Response Fields

  • product Product

    Newly created product

show

Get a paginated list of all the products that have been added by this organization to the specified feature

URL

GET /products/:feature

Required Parameters

  • feature String

    The feature e.g 'keyword'

Response Fields

  • products Array<Products>

    List of products that have been added by this organization to the specified feature

  • price BigDecimal
