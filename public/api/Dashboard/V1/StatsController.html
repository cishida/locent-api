Stats

dashboard

Gets stats for dashboard analytics page

URL

GET /dashboard/stats/dashboard/:from/:to

Required Parameters

  • from DateTime

    The lower boundary of the timeframe to get stats for. (in milliseconds since Epoch)
  • to DateTime

    The upper boundary of the timeframe to get stats for. (in milliseconds since Epoch)

stats

Gets stats for any feature's analytics page

URL

GET /dashboard/stats/:feature/:from/:to

Required Parameters

  • feature String

    The feature e.g 'keyword'
  • from DateTime

    The lower boundary of the timeframe to get stats for. (in milliseconds since Epoch)
  • to DateTime

    The upper boundary of the timeframe to get stats for. (in milliseconds since Epoch)
