Campaigns [1.0]

Campaigns

alert

Sends a message to all customers opt-ed in to a feature

URL

POST /dashboard/campaigns/alert/:feature

Required Parameters

  • feature String

    The feature e.g 'keyword'
  • message String

    The message to be sent
  • campaign_name String

    The name of the campaign

Response Fields

  • campaign Campaign

    Newly created campaign

import

Sends a message to all imported customers that are opted-in to any of the organization's features

URL

POST /dashboard/campaigns/import

Required Parameters

  • customers Array<Customer>

    The customers to be texted
  • message String

    The message to be sent
  • campaign_name String

    The name of the campaign

Response Fields

  • campaign Campaign

    Newly created campaign
  • failed_count failed_count

    The amount of customers that couldn't be sent to because they're not opted-in to any of the organization's features

index

Get a paginated list of organization's campaigns

URL

GET /dashboard/campaigns

Response Fields

  • campaigns Array<Campaigns>

    All organizations campaigns
