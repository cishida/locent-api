POST /dashboard/organizations/:id

Required Parameters

  • id Integer

    ID of the organization

Response Fields

create

Creates new organization

URL

POST /dashboard/organizations

Required Parameters

  • name String

    The organization name
  • email String

    The organization's email address
  • phone String

    The organization's contact phone number
  • first_name String

    The organization's primary user's first name
  • last_name String

    The organization's primary user's last name
  • password String

    The organization's primary user's password

Response Fields

  • organization Organization

    Newly created organization

  • phone String

    The organization's contact phone number

Response Fields

POST /dashboard/organizations/create_users

Required Parameters

    The users to be added to the organization

Response Fields

Change status of user from admin to normal or vice versa

URL

PUT /dashboard/organizations/update_user_admin_status

    The user's admin status

Response Fields
